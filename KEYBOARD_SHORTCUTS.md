# Barako Tour - Live Match Keyboard Shortcuts

## Score Controls

| Key | Action |
|-----|-------|
| **Q** | Increment Player 1 Score |
| **A** | Decrement Player 1 Score |
| **E** | Increment Player 2 Score |
| **D** | Decrement Player 2 Score |

## Race To Controls

| Key | Action |
|-----|-------|
| **+** (or **Shift + =**) | Increase Race To |
| **-** (or **_**) | Decrease Race To |

## Turn Indicator Controls

| Key | Action |
|-----|-------|
| **Tab** | Toggle Turn (Player 1 â†” Player 2) |
| **Z** | Set Turn to Player 1 (Left Arrow) |
| **C** | Set Turn to Player 2 (Right Arrow) |
| **X** | Clear Turn Indicator (No Arrow) |

## Ball Controls

| Key | Action |
|-----|-------|
| **1** | Toggle Ball 1 (Pocket/Unpocket) |
| **2** | Toggle Ball 2 (Pocket/Unpocket) |
| **3** | Toggle Ball 3 (Pocket/Unpocket) |
| **4** | Toggle Ball 4 (Pocket/Unpocket) |
| **5** | Toggle Ball 5 (Pocket/Unpocket) |
| **6** | Toggle Ball 6 (Pocket/Unpocket) |
| **7** | Toggle Ball 7 (Pocket/Unpocket) |
| **8** | Toggle Ball 8 (Pocket/Unpocket) |
| **9** | Toggle Ball 9 (Pocket/Unpocket) |
| **0** | Toggle Ball 10 (Pocket/Unpocket) |
| **Delete** | Reset All Balls (Unpocket All) |

## Reset Controls

| Key | Action |
|-----|-------|
| **R** (Double Press) | Reset Scores & Turn Indicator |

---

**Note:** Keyboard shortcuts are disabled when typing in input fields, text areas, or select dropdowns.
